# Premium Personal Portfolio

A highly polished, responsive personal portfolio website designed for **Amanulhaq**. This project showcases academic achievements, technical skills, and a comprehensive list of completed web development projects through a modern, multi-page user experience.

## ✨ Features

-   **Premium Design System**: A sophisticated "Deep Charcoal & Mint" theme featuring glassmorphism and subtle noise textures for a high-end, professional feel.
-   **Multi-Page Architecture**: Organized into dedicated pages for Home, About, Skills, Projects, Certifications, and Contact.
-   **Advanced Typography**: Utilizes a premium font pairing of **Outfit** (Headers) and **DM Sans** (Body) for maximum impact and readability.
-   **Dynamic Animations**: Powered by JavaScript `IntersectionObserver` for smooth entrance reveals and interactive hover effects.
-   **Fully Responsive**: Optimised for all screen sizes, from mobile devices to ultra-wide monitors.
-   **Vertical Centering**: Intelligent layout system that ensures content is perfectly centered on all screen sizes.

## 🛠️ Technology Stack

-   **Structure**: HTML5 (Semantic)
-   **Styling**: CSS3 (Custom Variables, Grid, Flexbox)
-   **Interactivity**: Vanilla JavaScript (ES6+)
-   **Icons**: Font Awesome 6.4
-   **Typography**: Google Fonts (Outfit & DM Sans)

## 🚀 Getting Started

1.  **Clone/Download** the repository to your local machine.
2.  Ensure your images (e.g., `portfolioo1.jpg.jpeg`, `resume1.jpeg`, `AMAZON11.jpeg`) are placed in the root directory.
3.  Open `index.html` in any modern web browser to view the site.

## 📝 Customization

-   **Colors**: You can easily change the theme color by updating the `--primary` variable in `style.css`.
-   **Fonts**: Fonts are managed via Google Fonts links in the `<head>` of each HTML file and the `font-family` property in `style.css`.

## 🚶‍♂️ Site Walkthrough

Experience a guided tour of the portfolio structure:

### 1. Home (`index.html`)
The landing page features a split-screen layout with a high-impact visual on the left and a professional introduction on the right. It provides immediate access to the project portfolio and resume.

### 2. About Me (`about.html`)
A dedicated space for a professional biography. It highlights core development principles and academic statistics (CGPA, Project Count) with a perfectly aligned profile photo.

### 3. Technical Skills (`skills.html`)
A visual grid of technical competencies, including programming languages (C++, Java, Python, JS) and tools (Git, GitHub, Bootstrap), each with interactive hover states.

### 4. Featured Projects (`projects.html`)
The core of the portfolio. Six detailed project cards featuring:
-   **Amazon Clone**: E-commerce functionality.
-   **Aurevix-Tech**: Modern agency UI.
-   **Expense Tracker**: Real-time financial management.
-   **Weather App**: API-driven data visualization.
-   **To-Do List**: Persistent task management.
-   **Portfolio**: The current high-end site design.

### 5. Certifications (`certifications.html`)
Showcases workshops and academic excellence, including Ethical Hacking and AI Tool certifications.

### 6. Contact (`contact.html`)
A vertically centered communication hub with direct email/phone links and a clean, responsive inquiry form.
