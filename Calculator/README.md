# Nova Calcify ✦
### Premium Glassmorphism Computation Experience

Nova Calcify is a high-end web-based calculator that redefines the modern arithmetic experience. Merging futuristic glassmorphism aesthetics with high-performance logic, it offers a tactile and visually stunning interface designed for creators, developers, and visionaries.

---

## ✦ Key Features

- **Luxury Intro Page**: A sophisticated landing experience featuring a 3D preview and premium marketing copy.
- **Tactile 3D Buttons**: Custom-engineered buttons that simulate a physical "click" with realistic depth and shadowing.
- **Intelligent Real-time Computation**: Results update instantly as you type, ensuring a fluid workflow.
- **Full Keyboard Integration**: Complete mapping for all arithmetic operations, allowing for lightning-fast input.
- **High-End Glassmorphism**: Utilizing advanced CSS backdrop filters and multi-layered radial gradients for a "frosted glass" look.
- **Responsive & Optimized**: Seamless performance across mobile, tablet, and desktop devices.
- **Obsidian Dark Aesthetic**: A deep, premium dark theme with neon blue and purple glowing accents.

---

## ✦ Tech Stack

- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Custom Design System, Flexbox, CSS Grid)
- **Logic**: ES6+ JavaScript (Class-based state management)
- **Typography**: Inter (Google Fonts)

---

## ✦ Getting Started

To experience Nova Calcify locally:

1.  **Clone or Download** the repository to your local machine.
2.  Locate the `index.html` file in the root directory.
3.  **Open `index.html`** in any modern web browser (Chrome, Safari, Firefox, Edge).
4.  Interact with the Intro Page and click **"Launch Calculator"** to begin.

---

## ✦ Keyboard Shortcuts

| Key | Action |
| :--- | :--- |
| `0-9` | Input Numbers |
| `.` | Input Decimal |
| `+`, `-`, `*`, `/` | Arithmetic Operations |
| `Enter` or `=` | Calculate Result |
| `Backspace` | Delete Last Digit |
| `Escape` | All Clear (AC) |
