# 🌌 Floating Layer Gallery — Premium Image Experience

> **"Transporting souls to a world of creativity."**

The **Floating Layer Gallery** is a high-end, immersive web application designed for fine art collectors and photography enthusiasts. It combines modern glassmorphism aesthetics with smooth, scroll-driven animations to create a "floating" interface that feels alive and premium.

---

## ✨ Key Features

-   **🎭 Floating Grid Layout**: An uneven, staggered grid that breaks the traditional "box" layout, giving each masterpiece its own weight and presence.
-   **🎨 Dynamic Category Filtering**: Seamlessly filter between *Nature*, *Architecture*, and *Abstract* collections with staggered re-entrance animations.
-   **🌑 Cinematic Lightbox**: A high-contrast, immersive modal for deep-diving into high-resolution images, featuring:
    -   Fluid scale-in transitions.
    -   Smooth keyboard navigation (Arrow keys).
    -   Minimalist typography overlays.
-   **✨ Glassmorphism UI**: Built with a sophisticated design system featuring frosted glass surfaces, soft ambient glows, and golden accents.
-   **🚀 Scroll-Driven Entrance**: Using the `IntersectionObserver` API, gallery items reveal themselves with a staggered "float-up" effect as you explore.
-   **📱 Fully Responsive**: A fluid design that adapts from high-end ultrawide monitors to compact mobile screens without losing its aesthetic charm.

---

## 📐 Design Philosophy

The project was built on the principle of **"Visual Weight and Depth."** Unlike standard galleries, the Floating Layer Gallery uses:
-   **Staggered Y-Offsets**: To create a sense of vertical depth.
-   **Radial Ambient Glows**: To simulate a high-end physical gallery lighting.
-   **Golden Accents (`#c9943e`)**: To provide a sense of luxury and focus.
